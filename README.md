# Lista de Tarefas (To-Do List)

Aplicação simples de gerenciamento de tarefas desenvolvida para praticar JavaScript e manipulação do DOM.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## Funcionalidades

- Adicionar novas tarefas
- Remover tarefas clicando no botão “Remover” em cada item
- Interface simples e intuitiva

## Como executar o projeto

1. Baixe ou clone o repositório
2. Abra o arquivo `index.html` no navegador

## Objetivo do projeto

Este projeto foi desenvolvido para praticar conceitos básicos de desenvolvimento front-end e lógica de programação.
